# Hey, I'm moose-lab.

Independent builder shipping AI-native tools for developer workflows, sports intelligence, and creative production. I mostly work in TypeScript, Python, and Next.js.

My current attention is on harnessing agents and AI models: making them observable, composable, and useful inside real building workflows.

**Current focus:** agent harnesses, model workflows, Sports AI surfaces, and creative production systems.

## Builder Notes

- I care about agent-native software that produces inspectable artifacts, not black-box demos.
- I like model workflows that can be prompted, audited, retried, and reused.
- I like tools that turn messy creative, sports, or coding work into repeatable, reviewable systems.
- Most experiments start small, ship quickly, and get tightened through real usage.
